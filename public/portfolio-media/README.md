# Portfolio Media Template

Put portfolio images in these folders and reference them from `src/data/portfolio.ts`.

Example:

```ts
media: {
  src: "/portfolio-media/projects/drowsiness-demo.jpg",
  alt: "Driver drowsiness detection demo",
  caption: "MediaPipe face mesh prototype",
}
```

Suggested folders:

- `profile/` for portrait, personal workspace, or hero imagery.
- `experience/` for company, event, dashboard, ambassador, or teaching assistant visuals.
- `projects/` for screenshots, diagrams, demo frames, notebooks, dashboards, or model outputs.
- `education/` for campus, scholarship, coursework, or certificate imagery.
- `awards/` for certificates and competition documentation.

Keep filenames lowercase with hyphens, for example `maybank-dashboard.jpg`.
